# Pera CMS とは？

Pera CMS は、軽量でシンプルな PHP ベースのコンテンツ管理システムです。

- `*.txt` ファイルだけで記事を書く
- 単一ファイル`read.php` がパース・レンダリングを担当
- スキンは PHP + HTML + CSS で柔軟に定義可能
- 完全オープンソース(Apacheライセンス)・純国産
- 一切のDB不要、超軽量！

@skin 1

# 特徴

- Markdownライクな文法
- セクション分けとスキンの読み込みが可能
- 図やYouTube埋め込み、画像もOK
- 超軽量(約32KB)
